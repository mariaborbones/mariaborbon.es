# mariaborbon.es

Sitio web personal de María Borbonés. Plain HTML/CSS/JS, no build step, deployed on [Render](https://render.com) as a static site.

## Local preview

Open `index.html` directly in a browser, or serve it locally:

```
npx serve .
```

## Deploy

Deployed via `render.yaml` as a Render static site. Pushing to the connected branch triggers a deploy automatically.
